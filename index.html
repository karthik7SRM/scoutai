Datasets

Football Dataset

Dataset consist of Top 5 League Player data. Player Name, Age,Nation, Position, Squad, Competition, Matches, Minutes, Goals, Assists, Goals and Assists, Number of Yellow Card and Red Cards

T20I DATASET

Dataset consist of Performance of Players in T20I format. Player Name, Country, Span, Number Of Matches, Number of Innings, Number of Not Outs, Runs, High Scores, Average, Balls Faced, Strike Rates, Centuries, Fifties, Ducks, Fours and Sixes

Test Dataset

Dataset consist of Performance of Players in TEST format. Player Name, Country, Span, Number Of Matches, Number of Innings, Number of Not Outs, Runs, High Scores, Average, Balls Faced, Strike Rates, Centuries, Fifties, Ducks, Fours and Sixes

ODI Dataset

Dataset consist of Performance of Players in ODI format. Player Name, Country, Span, Number Of Matches, Number of Innings, Number of Not Outs, Runs, High Scores, Average, Balls Faced, Strike Rates, Centuries, Fifties, Ducks, Fours and Sixes
